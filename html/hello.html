Description:

The Staleks Helper extension is designed to enhance the user experience when interacting with the website of Staleks company. Simplify content-related processes, maintain usability, and overcome inconveniences associated with copy protection and frequent SMS authentication.

This extension can do the following:

  • Adds a title copy button
  • Adds a produc id copy button
  • Adds a description copy button
  • Adds the ability to insert a search query from the buffer into the search menu

Hot keys:

  • Title copy (Ctrl+Up)
  • Produc id copy (Ctrl+Down)
  • Description copy (Ctrl+Right)
  • Insert a search query (Ctrl+I)

Note:

Staleks Helper is developed by independent developers and is not officially supported by the Staleks company. Please ensure that the use of the extension complies with the terms of use of the Staleks website.
